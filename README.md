Overview:  
The Candestick Pattern Scanner is a Python-based tool designed to analyze historical stock and cryptocurrency data to identify key candlestick patterns, such as Bullish Engulfing and Bearish Engulfing. Candlestick patterns are widely used in technical analysis to predict potential market reversals or continuations, making this project a practical demonstration of combining programming with financial analytics.

Technologies Used:  
Python - primary language for data analysis and automation
pandas - efficient for data manipulation and analysis
NumPy - for numerical calculations and logical operations
yfinance - fetches historical stock and cryptocurrency data
mplfinance - for plotting high-quality candlestick charts

Key Features:  
**Automated Pattern Detection - Scans historical data for common candlestick patterns such as:**
- Bullish Engulfing
- Bearish Engulfing  
**Date Output - Generates a list of dates when patterns are detected for easy reference.**  
**Data Visualization - Plots candlestick charts for selected stocks or crypto, providing a visual represenation of detected patterns.**  
**Extensible Design - Can be easily extended to include additional candlestick patterns, moving averages, or other technical indicators.**  

Learning Outcomes:  
**Financial Analysis - Understanding candlestick patterns and their role in technical trading.**  
**Data Science Fundamentals - Fetching, cleaning, and analyzing time-series financial data.**  
**Python Programming - Applying loops, functions, and libraries to automate a real-world taskl.**  
**Data Visualization - Communicating complex financial patterns visually using charts.**  
  
***Potential Applications***  
*This tool is a foundation for more advanced trading analytics, portfolio management dash boards, or automated trading systems. While this project is educational and demonstrative, it illustrates how programming skills can be applied to real-world financial data analysis.*
